# PeerDrop — Decentralized Large-File Transfer Hub

> **PeerDrop** is a high-performance, decentralized file-sharing platform that enables seamless transfer of large files directly between devices. Leveraging WebRTC for peer-to-peer connections with STUN/TURN traversal, it delivers blazing-fast LAN transfers and reliable cross-network delivery — all without central servers or file-size limits.
